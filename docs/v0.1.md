# v0.1 - minigrep propre

## Objectif

sgrep <pattern> <file>

## Modules

- src/main.rs
- src/lib.rs
- src/config.rs
- src/search.rs

## À implémenter

- parse pattern + file
- lire le fichier
- chercher les lignes qui contiennent le pattern
- afficher les lignes trouvées
- retourner une erreur propre si le fichier n’existe pas

## Tests

- trouve une ligne
- trouve plusieurs lignes
- ne trouve rien
- fichier inexistant
